# vue-tkang

A Vue component plugin
包装Element-ui关于form，以及echarts和一些自己常用的组件

## 安装

```JS
npm install vue-tkang -S
```

## 使用

```js
// ES6
import vueTk from 'vue-tkang.js'
// require
var vueTk = require('vue-tkang')

Vue.use(vueTk)

// 或者直接使用script导入
<script src="./node_modules/vue/dist/vue-tkang.js"></script>
